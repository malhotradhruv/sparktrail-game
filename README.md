# SparkTrail: Photon Odyssey

Welcome to SparkTrail, the high-stakes CTF game where you guide a rogue photon through a fractured quantum network. In Photon Odyssey, players must overcome eleven escalating challenges to restore coherence to the nebula core and assemble the final flag.

## Storyline

After a reactor collapse in the orbital lab Photonisphere, the last remaining energy packet is fractured into fragments and scattered across a labyrinth of encrypted systems. As a data runner, you must trace the photon through reflection, deflection, merger, retracement, interference, and coherence operations to recover the lost signal.

## Game Flow

1. Solve the 11 challenges from `C1` through `C11`.
2. Collect partial flag fragments for each stage.
3. Bring the fragments to the final boss for Roman numeral ordered assembly.
4. Unlock the complete flag in the format `SPARK{...}`.

## Game Structure

- `challenges/`: Eleven challenge nodes (C1–C11) with unique puzzles.
- `assets/`: Image, audio, and data files used by the game.
- `docs/`: Rules, instructions, and reference materials.
- `scripts/`: Validation and helper scripts for checking solutions.

## Flag Format

- The final flag uses the format `SPARK{...}`.
- Partial fragments follow the same core style and often include hidden Roman numerals.

## Objectives

1. Learn the transformation rules in `docs/rulebook.md`.
2. Solve each challenge using the correct ruleset.
3. Collect challenge flags and assemble the final flag using Roman numeral ordering.
